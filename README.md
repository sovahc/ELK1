## Электронный Литературный Критик Научной Фантастики (ЭЛК/НФ)
Разбивает произведение на части по нескольку глав, выписывает заметки по каждому фрагменту.
Затем оценивает выжимку из текста (свои заметки) по заданному вручную набору критериев.

В конце работы выводит итоговую оценку.
- Пример хорошей оценки произведения +8/-2 (Проект Аве Мария)
- Пример плохой оценки +5/-5

Протестирован с локальной LLM Qwen3.6-27B
